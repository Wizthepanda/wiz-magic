1. Information We Collect

Google Account Data (via OAuth)

Scope: youtube.readonly

Purpose: We access only read-only YouTube watch history and engagement data (e.g., watched videos, likes, comments) strictly to verify quest completion and award XP on our platform.

No other permissions requested: We never request scopes that allow posting, editing, uploading, deleting content, or accessing private messages or subscriber data.

Additional Data Collected

  • Email address (from Google Sign-In): Used solely for account management
  • XP transaction and engagement records: Stored to support quest history and reward tracking

2. How We Use Your Data

  • XP and Rewards Management: Confirm quest completion and grant XP based on verified YouTube activity
  • Platform Experience: Manage user progress and unlock features
  • Security & Communication: Email address used for essential account operations (e.g., password resets)

We do NOT use data for marketing, advertising, profiling, or selling to third parties.

3. Data Storage, Retention & Deletion

Where We Store

  • All personal and engagement data is securely stored in Firebase Firestore
  • Data at rest is encrypted by Google; access is limited by strict Firebase Security Rules — only the authenticated user and authorized administrators can access it

How Long We Keep Data

  • Stored only as long as necessary to support quest tracking and XP history
  • After 12 months of user inactivity, we automatically delete the data

Data Deletion

  • Users may request immediate deletion at any time by emailing wizsparkles@gmail.com
  • Upon request, all data linked to the user's account is erased within 7 business days

4. Data Security Measures

  • Encryption: Enabled at all times when data is at rest and in transit
  • Access Control: Firebase Security Rules restrict access to the authenticated user and trusted admins only
  • Audit: Regular reviews ensure only necessary data is retained
  • Incident Response: In the event of a security breach, users are notified promptly and remedial measures are implemented

5. Third-Party Sharing

We do not sell or share user data with advertisers or data brokers.

Data is shared only with:

  • Google Cloud / Firebase — our secure data processor
  • Trusted service providers — under NDA, solely for operational support (e.g., hosting, monitoring)

We do not use data for advertising, profiling, or external analytics.
